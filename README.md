# LameBook
Lamebook is simple Web App make with Ruby on Rails. The system has the basic features of a social media like:

* Sign in
* Sign up
* Logout
* Create a post
* Like a post

The tecnologies behind this application are:

* Ruby: 3.0.4
* Rails: 7.0.3
* Tailwindcss:  2.0

Any others, you can see on Gemfile.
